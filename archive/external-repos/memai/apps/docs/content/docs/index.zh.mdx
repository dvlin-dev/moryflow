# 简介

Memai 是一个**记忆即服务**平台，为 AI 应用提供语义记忆存储和检索 API。让您的 AI 代理具备长期记忆能力、基于向量的语义搜索、实体提取和知识图谱功能。

## 什么是 Memai？

Memai 让 AI 应用能够：

- **存储记忆** - 通过自动向量嵌入持久化对话、事实和上下文
- **语义搜索** - 使用自然语言查询找到相关记忆
- **提取实体** - 自动识别和追踪人物、地点和概念
- **建立关系** - 创建连接实体的知识图谱

## 核心功能

| 功能 | 描述 |
|------|------|
| **语义搜索** | 使用最先进的嵌入技术进行基于向量的相似性搜索 |
| **实体提取** | 从记忆内容中自动识别实体 |
| **关系图谱** | 通过类型化关系连接实体 |
| **Webhooks** | 记忆事件的实时通知 |
| **多租户** | 按 API 密钥隔离数据 |

## 使用场景

### AI 助手
赋予您的 AI 助手记住过往对话、用户偏好和学习到的事实的能力。

### 客户支持
构建能够跨会话记住客户历史和上下文的支持机器人。

### 知识管理
创建具有语义理解能力的可搜索知识库。

### 研究工具
构建能够按上下文存储和检索信息的研究助手。

## 快速示例

```bash
# 创建一条记忆
curl -X POST https://api.memai.dev/v1/memories \
  -H "Authorization: Bearer mm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "用户偏好深色模式并使用 vim 快捷键",
    "userId": "user_123"
  }'

# 搜索记忆
curl -X POST https://api.memai.dev/v1/memories/search \
  -H "Authorization: Bearer mm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "用户的偏好是什么？",
    "userId": "user_123"
  }'
```
