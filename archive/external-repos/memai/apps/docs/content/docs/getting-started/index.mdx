# Getting Started

Welcome to Memai! This guide will help you get up and running quickly.

## Overview

Getting started with Memai involves three simple steps:

1. **Create an account** - Sign up at [console.memai.dev](https://console.memai.dev)
2. **Get your API key** - Generate an API key from the console
3. **Make your first API call** - Start storing and searching memories

## Prerequisites

- A Memai account
- An API key (starts with `mm_`)
- Any HTTP client (curl, Postman, or your favorite programming language)
