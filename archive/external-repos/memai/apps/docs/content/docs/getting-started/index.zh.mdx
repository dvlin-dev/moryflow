# 快速开始

欢迎使用 Memai！本指南将帮助您快速上手。

## 概述

开始使用 Memai 只需三个简单步骤：

1. **创建账户** - 在 [console.memai.dev](https://console.memai.dev) 注册
2. **获取 API 密钥** - 从控制台生成 API 密钥
3. **发起首次 API 调用** - 开始存储和搜索记忆

## 前置条件

- 一个 Memai 账户
- 一个 API 密钥（以 `mm_` 开头）
- 任意 HTTP 客户端（curl、Postman 或您喜欢的编程语言）
