# Note Polisher

Use this skill when users ask to clean up notes, rewrite rough drafts, or make writing concise.

## Workflow

1. Identify key points and remove repetition.
2. Keep structure simple: heading, bullets, optional action items.
3. Preserve factual meaning and avoid adding unsupported claims.

## Output

- Return plain markdown.
- Prefer short paragraphs and clear bullet points.
