# Chat Pane（PC）

## 范围

- 负责聊天区域布局、消息列表与输入框组合
- 集成任务系统：输入框上方悬浮任务面板

## 关键文件

- `index.tsx`：ChatPane 容器与数据协调
- `components/chat-footer.tsx`：输入区 + 悬浮任务面板
- `components/task-hover-panel.tsx`：任务悬浮面板 UI/交互
- `hooks/use-tasks.ts`：Tasks 数据拉取/订阅

## 设计约束

- 任务悬浮面板不进入消息流；贴合输入框顶部
- 点击展开列表；点击任务显示 inline 详情
- 结构保持模块化与单一职责
