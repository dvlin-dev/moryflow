# MoryFlow PC 发布说明

当前 PC 发布说明以仓库中的统一事实源为准：

- `/docs/design/moryflow/runbooks/pc-release-and-auto-update.md`
- `/docs/design/moryflow/runbooks/macos-code-signing.md`

- 客户端自动更新：`download.moryflow.com`
- 当前公开平台：macOS `arm64` / `x64`
- Windows 当前不作为公开下载承诺

如果需要修改官网、文档站或下载入口，不要继续参考旧的根级 `manifest.json`、`releases.moryflow.com` 或 R2-only 结构。
