# 下载安装

MoryFlow 当前公开下载只提供 **macOS（Apple Silicon）** 和 **macOS（Intel）**。

手动下载安装包与查看版本说明统一走 [GitHub Releases](https://github.com/dvlin-dev/moryflow/releases)。应用内自动更新同样直接从 GitHub Releases 获取。

## 系统要求

**macOS**

- macOS 12.0 或更新版本
- Apple Silicon 和 Intel 都支持
- 4 GB 以上内存

**Windows**

- 当前没有公开安装包
- 不要再参考旧资料里关于公开 Windows 下载的说法

## 怎么选对 Mac 版本

- **Apple Silicon**：M1、M2、M3、M4 以及更新的 Apple 芯片 Mac
- **Intel**：运行受支持 macOS 版本的 Intel Mac

如果不确定，打开 **苹果菜单 → 关于本机**，看里面显示的是 **芯片** 还是 **处理器**。

## 在 macOS 上安装

1. 下载 `.dmg`
2. 双击打开
3. 把 MoryFlow 拖到「应用程序」文件夹
4. 从「应用程序」里打开 MoryFlow

<Callout title="如果极少数情况下仍被 Gatekeeper 拦住">
  去「系统设置 → 隐私与安全性」里点「仍要打开」，或者右键应用后选择「打开」。
</Callout>

## 第一次打开

第一次打开 MoryFlow 时，它会让你选择一个笔记文件夹。

<Callout title="这个文件夹是什么？">
  这就是你的本地 vault。MoryFlow 会把笔记作为普通 Markdown 文件保存在这里，所以离开应用也能直接读取。
</Callout>

建议位置：

- `~/Documents/MoryFlow`
- 你已经在做备份的独立工作目录

如果你想用产品内置同步，后面再到 [Cloud Sync 设置](/zh/docs/settings/cloud-sync) 里配置；如果你想继续用外部云盘同步本地文件夹，看 [多设备同步](/zh/docs/advanced/sync)。
