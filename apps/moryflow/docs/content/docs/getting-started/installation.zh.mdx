# 下载安装

MoryFlow 当前公开下载仅提供 **macOS（Apple Silicon）** 和 **macOS（Intel）**。

手动下载安装包与查看版本说明统一走 [GitHub Releases](https://github.com/dvlin-dev/moryflow/releases)。应用内自动更新底层使用 `download.moryflow.com`，但不建议用户直接手填地址。

## 系统要求

**macOS**

- macOS 12.0 或更新
- M 系列芯片和 Intel 都支持
- 4GB 以上内存

**Windows**

- 当前公开安装包暂未开放
- 等 Windows 发布链路恢复后，官网与文档会重新提供下载入口

## 怎么选对 Mac 版本

- **Apple Silicon**：M1、M2、M3、M4 以及更新的 Apple 芯片 Mac
- **Intel**：较早的 Intel 芯片 Mac

如果不确定，打开 **苹果菜单 → 关于本机**，看里面显示的是 **芯片** 还是 **处理器**。

## macOS 安装

1. 下载 `.dmg` 文件
2. 双击打开
3. 把 MoryFlow 拖到「应用程序」文件夹
4. 从「应用程序」里打开 MoryFlow

<Callout title="如果极少数情况下仍被 Gatekeeper 拦住">
  去「系统设置 → 隐私与安全性」里点「仍要打开」，或者右键应用后选择「打开」。
</Callout>

## 第一次打开

打开 MoryFlow 后，它会让你选一个**文件夹**来存放笔记。

<Callout title="为什么要选文件夹？">
  MoryFlow 的笔记都是普通的 Markdown
  文本文件，就存在你指定的文件夹里。你可以用任何编辑器打开，也可以把这个文件夹放到云盘里同步。
</Callout>

**建议位置**

- **Mac**：`文稿/MoryFlow` 或 iCloud 云盘里

如果你想让其他设备也自动同步，把文件夹放到云盘目录里即可。详见 [多设备同步](/zh/docs/advanced/sync)。
