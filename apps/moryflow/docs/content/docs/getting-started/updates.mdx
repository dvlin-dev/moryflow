# Download & Updates

MoryFlow uses **GitHub Releases** as the single source for both manual downloads and in-app automatic updates.

## Public Download Entrypoints

- **GitHub Releases** is the official page for manual downloads and release notes
- The [download page](https://www.moryflow.com/download) on the website fetches version info from GitHub Releases

If you want to download a build manually, read release notes, or find an older version, always start from GitHub Releases.

## Current Public Status

- Public downloads are currently available for:
  - macOS (Apple Silicon)
  - macOS (Intel)
- Windows is not currently published

## How In-App Updates Work

MoryFlow follows this update flow:

1. The app checks for updates automatically (20 seconds after launch, then every 6 hours)
2. Updates are **downloaded automatically** by default
3. You decide when to **restart and install**

That means update checks and downloads are automatic, but **install** stays under user control. You can turn off automatic downloads in Settings if you prefer to download manually.

## Where to Change Update Settings

Inside the app:

1. Open **Settings**
2. Go to **General**
3. Toggle automatic downloads on or off

## If an Update Fails

Use this fallback order:

1. Open [GitHub Releases](https://github.com/dvlin-dev/moryflow/releases)
2. Download the matching installer manually
3. Install it over your existing app
