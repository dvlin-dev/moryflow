# Download & Install

MoryFlow currently ships public downloads for **macOS (Apple Silicon)** and **macOS (Intel)**.

Manual downloads and release notes live on [GitHub Releases](https://github.com/dvlin-dev/moryflow/releases). In-app automatic updates use `download.moryflow.com` behind the scenes.

## System Requirements

**macOS**

- macOS 12.0 or later
- Apple Silicon or Intel both work
- 4GB RAM or more

**Windows**

- Public installer is not available yet
- The website and docs will switch back to Windows downloads after the Windows release workflow is restored

## Choosing the Right Mac Build

- **Apple Silicon**: M1, M2, M3, M4, and newer Apple Silicon Macs
- **Intel**: older Intel-based Macs

Not sure which Mac you have? Open **Apple menu → About This Mac** and check whether it says **Chip** or **Processor**.

## macOS Installation

1. Download the `.dmg` file
2. Double-click to open it
3. Drag MoryFlow to the Applications folder
4. Open MoryFlow from Applications

<Callout title="If Gatekeeper still blocks the app">
  If macOS still blocks the app in a rare case, go to **System Settings → Privacy & Security** and
  choose **Open Anyway**, or right-click the app and select **Open**.
</Callout>

## First Launch

When you open MoryFlow, it asks you to choose a **folder** to store your notes.

<Callout title="Why choose a folder?">
  MoryFlow saves notes as regular text files (Markdown) in the folder you pick. You can open them
  with any editor, and you can put that folder in a cloud drive to sync across devices.
</Callout>

**Recommended location**

- **Mac**: `Documents/MoryFlow` or inside iCloud Drive

Put it in a cloud drive folder if you want automatic sync to other devices. See [Multi-device Sync](/docs/advanced/sync) for details.
