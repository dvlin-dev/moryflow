# 下载与更新

MoryFlow 把**手动下载**和**应用内更新分发**分开处理。

## 公开下载入口

- **GitHub Releases**：官方手动下载页和 Release Notes 页面
- **download.moryflow.com**：只给应用内更新 feed 使用

如果你想手动下载安装包、看版本说明，或者找历史版本，请统一从 GitHub Releases 开始。

## 当前公开状态

- 当前公开下载平台：
  - macOS（Apple Silicon）
  - macOS（Intel）
- Windows 当前没有公开发布
- 官网和文档站有时会优先展示 Beta 版本

## 应用内更新怎么工作

MoryFlow 当前采用下面这套流程：

1. 应用自动检查更新
2. 由你决定什么时候 **Download**
3. 由你决定什么时候 **Restart and install**

也就是说，**检查**是自动的，但**下载**和**安装**仍然由用户确认。

## Stable 和 Beta

MoryFlow 目前支持两个更新通道：

- **Stable**：默认长期路径
- **Beta**：更早拿到新版本，适合愿意提前验证的人

如果当前公开分发以 Beta 为主，官网和文档会明确标出来。

## 在哪里看更新设置

在应用里：

1. 打开 **Settings**
2. 进入 **General**
3. 在这里查看更新状态、下载动作和当前通道

后续如果界面文案再调整，底层规则也还是“自动检查 + 用户确认下载和安装”。

## 更新失败怎么办

按这个顺序兜底：

1. 打开 [GitHub Releases](https://github.com/dvlin-dev/moryflow/releases)
2. 手动下载对应安装包
3. 直接覆盖安装现有应用
