# 下载与更新

MoryFlow 把**手动下载**和**应用内自动更新分发**分开处理。

## 公开下载入口

- **GitHub Releases**：官方手动下载页与 Release Notes 页面
- **download.moryflow.com**：应用内更新 feed 使用，不作为普通下载页对外暴露

如果你想手动下载安装包、查看版本说明或找历史版本，请统一从 GitHub Releases 开始。

## 当前公开状态

- 当前公开下载通道是 **Beta**
- 当前公开平台：
  - macOS（Apple Silicon）
  - macOS（Intel）
- Windows 版本暂时没有公开发布

## 应用内更新怎么工作

MoryFlow 采用下面这套策略：

1. 应用自动检查更新
2. 由你手动确认 **Download update**
3. 由你手动确认 **Restart to update**

也就是说，**检查**是自动的，但**下载**和**安装**仍然由用户确认。

## Stable 和 Beta 的区别

- **Stable**：长期默认通道
- **Beta**：用于更早验证新版本
- 如果官网当前公开的是 beta，页面会明确标注为 Beta

## 在哪里切换更新通道

在应用里：

- 打开 **Settings**
- 进入常规/更新区域
- 在 **stable** 和 **beta** 之间切换

## 更新失败怎么办

按这个顺序兜底：

1. 打开 [GitHub Releases](https://github.com/dvlin-dev/moryflow/releases)
2. 手动下载对应安装包
3. 直接覆盖安装现有应用
