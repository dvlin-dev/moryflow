# 下载与更新

MoryFlow 统一使用 **GitHub Releases** 作为手动下载和应用内自动更新的唯一源。

## 公开下载入口

- **GitHub Releases**：官方手动下载页和 Release Notes 页面
- 官网[下载页](https://www.moryflow.com/zh/download)从 GitHub Releases 获取版本信息

如果你想手动下载安装包、看版本说明，或者找历史版本，请统一从 GitHub Releases 开始。

## 当前公开状态

- 当前公开下载平台：
  - macOS（Apple Silicon）
  - macOS（Intel）
- Windows 当前没有公开发布

## 应用内更新怎么工作

MoryFlow 当前采用下面这套流程：

1. 应用自动检查更新（启动后 20 秒，之后每 6 小时一次）
2. 默认 **自动下载** 更新
3. 由你决定什么时候 **重启安装**

也就是说，检查和下载都是自动的，但**安装**仍由用户确认。你可以在设置中关闭自动下载，改为手动下载。

## 在哪里看更新设置

在应用里：

1. 打开 **Settings**
2. 进入 **General**
3. 在这里切换自动下载开关

## 更新失败怎么办

按这个顺序兜底：

1. 打开 [GitHub Releases](https://github.com/dvlin-dev/moryflow/releases)
2. 手动下载对应安装包
3. 直接覆盖安装现有应用
