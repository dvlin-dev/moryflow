# 常见问题

## 下载与更新

### 现在有公开 Windows 下载吗？

没有。当前公开下载只包括：

- macOS（Apple Silicon）
- macOS（Intel）

如果你要找当前官方构建或版本说明，请统一从 [GitHub Releases](https://github.com/dvlin-dev/moryflow/releases) 开始。

### 为什么 macOS 提示无法验证开发者？

极少数情况下，即使应用来自官方 release 页面，Gatekeeper 仍然可能拦住它。

可以这样处理：

1. 打开 **系统设置 → 隐私与安全性**
2. 选择 **仍要打开**
3. 或者右键应用后选择 **打开**

### 更新是怎么工作的？

MoryFlow 会自动检查更新，但下载和安装仍然需要你确认。

看 [下载与更新](/zh/docs/getting-started/updates)。

## 同步

### Cloud Sync 和外部文件夹同步有什么区别？

它们不是同一个功能。

- **Cloud Sync** 是 MoryFlow 的内置同步路径
- **外部文件夹同步** 是 MoryFlow 之外的一层文件同步

看 [多设备同步](/zh/docs/advanced/sync) 和 [Cloud Sync 设置](/zh/docs/settings/cloud-sync)。

### 为什么 Cloud Sync 现在不可用？

当前 Cloud Sync 依赖：

1. 你已经登录
2. 你已经打开了一个 vault

少任何一个，应用都会先停在那个状态，不会直接进入同步控制页。

## 账号与设置

### 去哪里管理会员和积分？

去 Settings 里的 **Account** 分区。

当前 MoryFlow 在这里处理：

- 登录
- 会员
- 积分
- 删除账号

看 [账号与会员](/zh/docs/settings/account-and-membership)。

### 去哪里配置 Providers 和模型？

去 Settings 里的 **Providers** 分区。

这里负责：

- 内置服务商
- 自定义服务商
- API Key
- 启用的模型
- 默认模型选择

看 [Providers 与 Models](/zh/docs/settings/providers-and-models)。

### Providers 和 MCP 有什么区别？

它们解决的问题不一样。

- **Providers** 决定 Mory 用哪家模型
- **MCP** 给 Mory 增加外部运行时能力

看 [Providers 与 Models](/zh/docs/settings/providers-and-models) 和 [MCP 设置](/zh/docs/settings/mcp)。

## AI 使用

### 为什么 Mory 没有回复？

先检查这几件事：

1. 网络连接
2. provider API key
3. 当前模型是否可用
4. 聊天区域或设置里有没有错误提示

### 可以完全离线用吗？

如果是 AI 工作流，当前离线路径是使用本地模型，比如 Ollama。

看 [Ollama 本地模型](/zh/docs/settings/ollama)。

不管怎样，你的 vault 仍然是 local-first，因为笔记本身就是本地普通文件。
