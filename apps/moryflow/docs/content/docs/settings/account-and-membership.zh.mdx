# 账号与会员

**Account** 分区负责身份、会员、积分和账号安全。

## 未登录时

如果你还没登录，这里会承载完整的账号流程：

- 邮箱密码登录
- Google 登录
- 邮箱注册
- 找回密码

注册不是一张长表单，而是当前这三步：

1. 输入邮箱
2. 用 OTP 验证邮箱
3. 设置密码

## 登录后

登录后，这一页会变成你的账号总览。

### 个人资料

这里可以查看：

- 头像
- 昵称
- 邮箱
- 邮箱验证状态
- 当前会员 badge

你也可以在这里编辑资料和退出登录。

### 积分

MoryFlow 当前会显示四类积分：

- Daily credits
- Subscription credits
- Purchased credits
- Total credits

这样你能更清楚地看到现在剩多少，以及是否需要补积分或升级会员。

### 会员

这一块会展示：

- 当前 tier
- 该 tier 对应的功能列表
- 是否还能继续升级

如果你当前还不是最高 tier，这里会提供升级入口。

### 积分包

除了会员，你也可以在 Account 里购买一次性的积分包。

积分包和订阅会员不是一回事。

## 找回密码

如果忘记密码，Account 里当前提供的是：

1. 输入邮箱
2. OTP 验证
3. 设置新密码

## 删除账号

删号流程现在是明确收紧的。

删除前必须完成：

1. 选择删除原因
2. 可选填写补充反馈
3. 输入当前账号邮箱做确认

## 这个分区不负责什么

Account **不负责**：

- 模型服务商
- MCP servers
- Cloud Sync 状态

这些能力分别放在别的设置分区里，避免把计费和运行时配置混在一起。
