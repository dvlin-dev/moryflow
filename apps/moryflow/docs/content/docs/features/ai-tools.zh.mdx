# Mory 运行时能力

这一页刻意保持高层描述。

旧文档喜欢用一份固定的内置工具清单来解释 Mory，但这种写法已经不够稳定，不适合继续当用户层的长期契约。

## 更好的理解方式

当 Mory 在处理任务时，它可能会用到这些能力类别：

- 当前工作区里的文件访问
- 笔记创建或编辑
- 本地内容搜索
- 可用时的 Web 访问
- 支持场景下的命令执行
- 通过 MCP 接入的外部能力

具体会走哪条运行时路径，取决于当前应用版本、你的 provider 配置、权限状态，以及已经连接的 MCP server。

## 稳定的部分是什么

稳定的不是一份精确工具清单，而是这条工作流：

1. 你给 Mory 一个请求
2. 需要时补上下文
3. Mory 会展示自己正在做什么
4. 多步骤工作会显示为任务进度
