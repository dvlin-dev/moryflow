# Mory

Mory is the AI working area inside MoryFlow. It is not just a place to chat. It is where notes, files, models, tools, and task progress come together.

## How Conversations Start Now

MoryFlow currently has two common entry patterns:

### Start from a Conversation

If you already know what you want, type directly into the chat input and send your request.

### Start from the Pre-thread Entry

If you are not sure how to begin, the pre-thread area helps you start with:

- guided starter tasks
- suggested skills
- prompt prefill instead of auto-send

That means the app helps you begin, but you still stay in control of the final prompt.

## What Mory Works With

Mory can work with several kinds of context in the same flow:

- your current note
- other files from your vault
- selected text
- uploaded attachments
- the model you chose
- any currently connected MCP capabilities

## Transparent Task Progress

When a task needs multiple steps, MoryFlow can show task progress in the chat area.

That panel is lightweight by design:

- no hidden background workflow model
- no separate task app
- just a visible summary of what the current conversation is doing

## Models and Access

Mory does not have one fixed intelligence source.

Its behavior depends on:

- which provider you configured
- which model is currently selected
- whether you enabled extra MCP capabilities
- what files or selections you gave it as context

That is why model setup belongs in **Settings**, not inside the notes themselves.

## What Mory Can Help With

Typical requests include:

- summarize notes
- turn rough notes into a plan
- rewrite an article
- compare ideas across files
- prepare a publishable page
- gather outside context when web access or MCP is available
